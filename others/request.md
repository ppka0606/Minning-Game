#### 总览
---
- 项目内容   
    - 用户登录后，从最低级别开始游戏。根据游戏级别不同,采用不同难度的迷宫地图,且有不同的过关时间限制; 在规定时间之内,找到迷宫出口,并挖出尽可能多的矿石; 不同的矿石可以增加过关时间或者增加财富, 在迷宫的每一个选择路口以概率提供可能的矿产信息由玩家选择. 当游戏时间不够时,可以使用时间矿石换取游戏时间, 游戏时间结束还未走出迷宫, 则game over, 否则进入下一关.要求采用键盘或者鼠标操作人物移动。可选功能：可以 提供玩家不同类型的人物选择,比如兔子速度快, 挖宝能力弱; 熊速度慢,但挖宝速度快; 人速度居中, 挖宝能力居中,但寻宝能力最强,且有一定的学习能力
- 设计语言及开发平台   
    - 初步选定python
    - 使用pygame工具进行开发
    - 基于Anaconda的python 3.7.6
    - 编辑器选择 Visual Studio Code
- 存储位置  
    - 源代码存储于/code/
        - 单元测试代码存储于/code/unittest/
        - 实际项目代码存储与/code/game/
    - 游戏相关资源存储于/resource/
    - 其他内容存储于/others/
