#### 总览
---
- 项目内容   
    - 在图形界面显示自动生成的具有可行通路的迷宫，用户控制键盘移动小人在迷宫中行走，找到出口则显示“您赢了”，用户登录后，可以随时保存游戏，退出游戏，进入未完成游戏和开始新游戏等功能。记录用户胜负次数并可以查看胜负次数。用户可以选择提示寻路路径，则电脑自动输出入口到出口的通路。
- 设计语言及开发平台   
    - 初步选定python
    - 使用pygame工具进行开发
    - 基于Anaconda的python 3.7.6
    - 编辑器选择 Visual Studio Code
- 存储位置  
    - 源代码存储于/code/
        - 单元测试代码存储于/code/unittest/
        - 实际项目代码存储与/code/game/
    - 游戏相关资源存储于/resource/
    - 其他内容存储于/others/
